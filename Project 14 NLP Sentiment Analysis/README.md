# 🎬 KGF 2 Sentiment Analysis Dashboard

A comprehensive NLP project analyzing movie reviews for **KGF: Chapter 2**. This application utilizes Hugging Face Transformers to perform sentiment analysis, text summarization, and machine translation on a custom dataset of audience reviews.

## 🚀 Features
* **Dynamic Sentiment Analysis:** Interactive gauge charts displaying classification confidence scores (DistilBERT).
* **Scaled Dataset:** Analyzes a custom-built, balanced dataset of KGF 2 reviews.
* **NLP Pipeline Showcase:** Demonstrates Machine Translation (English to Spanish) and Text Summarization.
* **Interactive UI:** Built entirely with Streamlit and Plotly for a seamless data visualization experience.

## 🛠️ Tech Stack
* Python
* Streamlit
* Hugging Face Transformers (Pipeline API)
* PyTorch
* Plotly
* Pandas

## 💻 Local Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `streamlit run app.py`
